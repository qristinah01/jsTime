# Drop your images here

Place these four files in this `public/images/` folder with these exact names:

- `portrait.jpg` — your hero portrait (used in Hero and About)
- `work-learning-galaxy.jpg` — Learning Galaxy case study cover (16:10 recommended)
- `work-bet-slip.jpg` — BET SLIP UX case study cover (16:10 recommended)
- `work-landing.jpg` — High-converting landing case study cover (16:10 recommended)

Export them from your Figma file at 2x resolution as JPG (portrait) or PNG (case studies).

**Recommended dimensions:**

- Portrait: 1840 × 2300 px minimum
- Case study covers: 1800 × 1125 px minimum

If you need different filenames, edit the paths in `components/Hero.tsx`, `components/About.tsx`, and `components/Work.tsx`.
